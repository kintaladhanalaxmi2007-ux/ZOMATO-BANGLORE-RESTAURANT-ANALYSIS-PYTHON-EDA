# ZOMATO-BANGLORE-RESTAURANT-ANALYSIS-PYTHON-EDA

This project analyzes the Zomato Bangalore restaurant dataset to understand food trends, customer preferences, restaurant ratings, pricing patterns, and location-based distribution. The goal is to generate insights that help restaurant owners and Zomato make better business decisions.

Business Problem 

The objective of this project is to analyze restaurant data in Bangalore and identify factors that influence restaurant success. 

The analysis helps to:

Identify popular cuisines Understand pricing patterns Find high-performing locations Analyze the impact of services like online ordering and table booking

Dataset Dataset: Zomato Bangalore Restaurants Dataset

https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants⁠ The dataset contains information about restaurants listed on Zomato in Bangalore.

Key Features Restaurant Name Online Order Availability Table Booking Availability Rating Votes Location Restaurant Type Cuisines Approx Cost for Two Service Type (Delivery / Dine-out) Tools & Technologies Used Python Pandas NumPy Matplotlib Seaborn

Jupyter Notebook 

Data Preprocessing The dataset contained missing values and inconsistent formats.

The following preprocessing steps were performed:

Cleaning the rating column Converting cost column to numeric Handling missing values Removing duplicate records Splitting and exploding cuisine data for analysis Exploratory Data Analysis Several analyses were performed to understand restaurant trends in Bangalore: Univariate Analysis Distribution of ratings Cost for two people Number of votes Bivariate Analysis Rating vs Online Ordering Rating vs Table Booking Rating vs Restaurant Type Rating vs Location Restaurant Distribution Delivery vs Dine-out restaurants Most popular locations Most common cuisines

Key Insights

Restaurants offering online ordering tend to receive higher customer engagement.

Mid-range restaurants attract more customers compared to very expensive ones.

Certain cuisines dominate the Bangalore food market.

Some locations have a high concentration of restaurants and customer activity.

Business Recommendations

Promote restaurants that provide online ordering services.

Focus on mid-range pricing strategies to attract more customers. 

Expand partnerships in high-demand locations.

Encourage restaurants to offer popular cuisines preferred by customers.

Conclusion 

This analysis helps understand the restaurant ecosystem in Bangalore and provides valuable insights that can support business decisions for restaurant owners and food delivery platforms like Zomato.
